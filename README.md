# Loopr Desktop Releases

Canal público oficial dos instaladores do Loopr Desktop. Este repositório contém somente artefatos de release e metadados de atualização; o código-fonte do aplicativo não é publicado aqui.

## Plataformas

- **Windows 10/11 x64:** instalador NSIS sem Authenticode e auto-update pelo próprio aplicativo.
- **macOS Monterey 12+ Apple Silicon:** download manual do DMG arm64.
- **macOS Monterey 12+ Intel:** download manual do DMG x64.

O Windows pode exibir um aviso do Microsoft Defender SmartScreen porque o instalador não possui certificado Authenticode. Confirme que o download veio desta página oficial antes de continuar.

Os pacotes macOS usam assinatura ad-hoc e não são notarizados pela Apple. Na primeira abertura, baixe apenas desta página oficial, mova o Loopr para Aplicativos, clique com Control/right-click no app e selecione **Abrir**. O macOS também pode oferecer **Abrir Mesmo Assim** em Ajustes do Sistema > Privacidade e Segurança.

Use somente releases publicadas. Rascunhos e artefatos intermediários não fazem parte do canal de distribuição.
